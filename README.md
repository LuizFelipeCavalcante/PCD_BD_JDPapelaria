# 📚 Projeto em Ciência de Dados: Banco de dados

Este é um projeto acadêmico desenvolvido no 2º período do curso de **Ciência de Dados**.  
O objetivo é criar um **banco de dados** e um **sistema desktop** para auxiliar na gestão de uma papelaria.

## 🎯 Objetivos
- Criar um banco de dados para armazenar informações de clientes, fornecedores, produtos, vendas e estoque.  
- Desenvolver um sistema desktop em **Python** para interação com o banco de dados.  
- Permitir consultas, cadastros, atualizações e relatórios de forma simples e eficiente.  

## 🛠 Tecnologias previstas
- **Python** (interface gráfica e lógica do sistema)
- **Qt Designer** (criação da interface gráfica)
- **SQLite** (banco de dados leve e integrado ao sistema)  

## 🚀 Funcionalidades previstas
- Cadastro de clientes, fornecedores e produtos.  
- Registro de vendas e controle de estoque.  
- Relatórios de vendas e movimentação de produtos.  
- Interface simples e intuitiva para uso em desktop.
